## Cách chạy dự án (Windows)

1. Cài **Node.js** bản mới (tải từ `https://nodejs.org` → cài đặt xong đóng lại).

2. Mở thư mục dự án này (`hiss`) bằng **VS Code** hoặc **Cursor**.

3. Mở **Terminal** trong VS Code/Cursor:
   - Menu `View` → `Terminal`.

4. Cài thư viện (chỉ làm 1 lần):

   ```bash
   npm install
   ```

5. Chạy cả trang web và dữ liệu:

   ```bash
   npm run dev
   ```

6. Mở trình duyệt và dán địa chỉ:
   - `http://localhost:5173`

7. Khi không dùng nữa:
   - Quay lại cửa sổ Terminal đang chạy
   - Nhấn `Ctrl + C` → nhấn `Y` (nếu được hỏi) để dừng.
